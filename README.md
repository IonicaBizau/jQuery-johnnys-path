jQuery-johnnys-path
==================

A small jQuery plugin that animates an absolute positioned element according to a path you give.

## How to use?

```js
var options = {};
$("...").johnnysPath(options, [
  { x: 100, y: 0   },
  { x: 100, y: 100 },
  { x: 000, y: 100 },
  { x: 0,   y: 0   }
]);
```
